    <h2>How to Play</h2>
    <p>The goal of the game is to guide the player character, Rando, through a series of levels and reach the end goal. Rando can move left and right, jump, and attack enemies using the arrow keys and the space bar. The game also includes power-ups and collectibles that the player can collect to gain special abilities or points.</p>
    <h2>Project Structure</h2>
    <p>The project is organized as follows:</p>
    <ul>
      <li>`assets/`: contains art assets and other resources used in the game</li>
      <li>`scripts/`: contains GDScript files that define the game mechanics and behaviors</li>
      <li>`scenes/`: contains Godot scenes that define the structure and layout of the levels.</li>
    </ul>
    <h2>Requirements</h2>
    <p>To play this game, you will need to install the Godot game engine on your computer. The game was developed and tested with Godot version 3.2.3, but it should also work with newer versions of the engine.</p>
    <h2>Technologies Used</h2>
    <ul>
      <li><a href="https://docs.godotengine.org/en/stable/getting_started/scripting/gdscript/index.html">GDScript</a>: a high-level, dynamically-typed programming language used to create scripts and behaviors in Godot</li>
      <li><a href="https://docs.godotengine.org/en/stable/getting_started/step_by_step/scenetree_and_nodes.html">Godot Scene Description Language (GSDL)</a>: a declarative language used to define the structure and layout of Godot scenes</li>
      <li><a href="https://docs.godotengine.org/en/stable/index.html">Godot Engine 3.5</a></li>
    </ul>
